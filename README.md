This project aims to explore the techniques for diffusion models together with mathematical
ideas behind it.

The implemented package `diffusion_playground` provides functionality to create simple
diffusion techniques like the forward process (incl. visualization) and training
simple models for the backward process and generating new data.

besides the implementation, the mathematical foundation is a focus of this work, aiming
to understand and apply the theoretical groundwork. For this, `README.md` files can
be found in the packages, outlining usage as well as background of the
modules.

## Installation ğŸš€

To install the project, follow the steps outlines below. Note that this is a basic Python
setup and can therefore easily be extended to further package manager etc.

1. Clone the repository
2. Create a virtual environment for Python

```bash
python3 -m venv .venv
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

With these steps, the setup is completed and the given scripts as well as notebooks
can be executed.
